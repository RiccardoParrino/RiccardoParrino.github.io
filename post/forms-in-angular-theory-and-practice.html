Forms in Angular

#angular #forms #user #interaction #validation #frontend #development

Introduction

Applications use forms to enable users to log in, to update a profile, to enter sensitive information, and to perform many other data-entry tasks. Angular provides two different approaches to handling user input through forms: reactive and template-driven. Both capture user input events from the view, validate the user input, create a form model and data model to update, and provide a way to track changes. In general, it is recommended to use the reactive driven approach to make forms, but you can evaluate what is the best method for your scenario.
