Introduction to JavaScript

#javascript #basics #web #development

Introduction

With this tutorial you will start understanding and writing something about JavaScript. Alongside with CSS3 and HTML5, JavaScript is most basic language for making website and web app. If HTML5 structure the content of the page and CSS3 gives it the style beautifing it, JavaScript make it dynamic and responsive to user interaction. JavaScript learning curve is enough flat, but knowing all of the details about async code, event handling and so forth makes you stand out from the crowd. It can get time, but it's worth doing it.

Prerequisites

Like we've said for CSS3 tutorial, it's better to know about all of the other cornerstone of the Web development. In this case, knowing about HTML5 and CSS3, make yourself complete and ready to learn about JavaScript.

Conclusion

With this tutorial you've covered most of the basic topic of JavaScript and you're ready to develop something interesting. From this point, you can go for some JavaScript framework, like Angular, Vue or React. This will simplify the development of your webapp, like Bootstrap or Tailwind for CSS3.
