Introduction to TypeScript

#typescript #type #javascript #frontend #development

Introduction

In this little tutorial, you will know about TypeScript and the difference from JavaScript. Anyway, if you're interested to develop web application with Angular, it's highly recommended to know about this language. Also TypeScript make it simple to work with type, instead of having deal with the weak typing of JavaScript. Furthermore, if you come from language like Java, C# or also Kotlin, you can appreciate the generic programming that TypeScript offer to you for work with.

Prerequisites

For this tutorial is necessary having some knowledge of JavaScript. It's not strictly required, you can come also from Java, Python or some other language, but the transition from JavaScript to TypeScript is quite natural.
