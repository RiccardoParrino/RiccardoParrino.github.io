Advanced Component in Angular

#angular #advanced #component #frontend #development

Introduction

This tutorial is directly extracted from the ngBook book of Nate Murray. For this section, you will dive deep into advanced features that you can use to make better components. With this tutorial you'll see how to styling components, modifying host DOM elements, modifying templates with content projection, accessing neighbor directives, using lifecycle hooks and detecting changes.

Prerequisites

For this tutorial it's necessary to know a good bunch of thing about Angular to understand it at a good level. You should know about how Angular works, Angular Component, Input and Output between Component, User interaction with component and a little bit of RxJs.

Conclusion

What you see in this little lesson let you know how to better managing component as styling, lifecycle and interaction with the other component. After understanding all of the topics and all of the details at the deepest level, outlined here, you will able to render any kind of component, also with regards to performance and optimization.
