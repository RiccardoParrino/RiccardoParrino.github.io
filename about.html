Summary

Hi, I am Riccardo and I am a software developer! I'm based in Palermo, Sicily and I'm currently looking for a new job! I've studied to University of Palermo where I got my Master's Degre in 2023. My Master's Thesis was awesome: I found a new method for Skin Mole Detetion in Total-Body Dermatological Images. This was a useful tool for melanoma detection and patient follow up analysis in determatological settings. For this application, I've used Computer Vision and Machine Learning with the support of Matlab. After my master thesis I started to work as a Java Backend Developer in a startup doing some work in Spring and Spring Boot for a company in the IT healthcare market. On March of 2025, I left my job and I start studying to became full stack developer. I've studied HTML5, CSS3 and JS as a foundational knowledge and then I started making some stuff in Angular hoping to find a new job with this position. At the same time I started this blog, posting what I was doing during in the period. If you want, you can get the resources I've posted for free. You can find some slides on HTML, CSS, JS and Angular, but also about marketing and information management system, hoping to start a my own company some day... In this page you can find about the timeline of my jobs, the technical skills I've acquired and my educational achievement. As I've said, I'm looking for a new job and if you want, you can contact me using the button at the end of the page.
